Hospitality Domain Data Analysis Project 🏨📊

Overview
This project analyzes hospitality industry data to uncover insights related to bookings, revenue, occupancy, and hotel performance. The dashboard helps stakeholders monitor key performance indicators and make data-driven decisions.

Key Focus
My primary contribution in this project was:
1. Creating DAX measures
2. Building interactive Power BI dashboards
3. Developing hospitality KPIs and performance metrics

Key Metrics Implemented
Some important business metrics created using DAX include:
Revenue
Total Bookings
Total Capacity
Successful Bookings
Occupancy %
Average Rating
Cancellation %
No Show Rate %
ADR (Average Daily Rate)
RevPAR (Revenue per Available Room)
Realisation %
DBRN, DSRN, DURN
Advanced Metrics
Revenue WoW Change %
Occupancy WoW Change %
ADR WoW Change %
RevPAR WoW Change %
Realisation WoW Change %
DSRN WoW Change %

Tools & Technologies
1. Power BI
2. DAX (Data Analysis Expressions)
3. Data Modeling
4. Dashboard Design
5. Dashboard Features
6. Interactive KPI cards

Weekly performance comparison
1. Platform-wise booking distribution
2. Room class booking analysis
3. Revenue and occupancy performance tracking

Outcome
The dashboard provides a clear view of hotel performance, enabling better monitoring of revenue trends, occupancy rates, booking behavior, and operational efficiency.
